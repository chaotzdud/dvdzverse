# DVDZVERSE - Estúdio de Soluções Digitais

Repositório do código-fonte do portfólio digital **DVDZverse**.

**Acesse o site:** [**www.dvdzverse.com.br**](https://www.dvdzverse.com.br)

## Sobre o Projeto

A **DVDZverse** é um estúdio de soluções digitais que integra três unidades de negócios especializadas, oferecendo serviços completos em tecnologia, design e produção musical. O site se posiciona como um hub criativo multidisciplinar, dividido em três portfólios principais:

* [**DVDZ Tech**](https://www.dvdzverse.com.br/tech) (Programação e Desenvolvimento)
* [**DVDZ Design**](https://www.dvdzverse.com.br/design) (Design Digital e UI/UX)
* [**DVDZ Beatz**](https://www.dvdzverse.com.br/beats) (Beatmaking e Produção Musical)

## Como Rodar o Projeto Localmente

```bash
# 1. Clone o repositório
git clone [https://github.com/chaotzdud/dvdzverse.git](https://github.com/chaotzdud/dvdzverse.git)

# 2. Navegue até o diretório
cd dvdzverse

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
```
